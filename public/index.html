Edit Mode — admins only
When editing is on, click any plain data cell (not the checkmark/badge columns, and not the Physchem or Flagged Rows tables) to edit it directly — each change saves to the server automatically and everyone sees it on their next load. Export still downloads a local JSON copy if you want one. Reset All Edits discards every admin edit and restores the last Excel sync, for everyone.
